1、 物理按钮
   - 短按 ：进入/退出低功耗模式
   - 长按3s松开后 : 则蓝牙恢复出厂

2、指示灯（蓝/绿/红）
         - 熄灭：关机（app上操作），或电池已充满电；
         - 蓝色灯闪烁：蓝牙未连接
         - 绿色灯闪烁：工作状态
        - 红色闪烁：电量不足，请尽快为设备充电
        - 红色常亮：设备正在充电中

3、蓝牙APP功能
   - 开关：控制设备软件开关，软件开机后才可以进入工作模式
   - 充电状态展示：是否在充电中
   - 电量展示
   - 游戏模式设置及展示
       - 快/fast
          - 正转7.2S；停止5.2S，反转5.2S，停止3S，正转3S，停止7.2S，反转7.2S，停止3S，正转3S，停止5.2S，反转7.2S，停止7.2S，依此循环工作。
       - 慢/slow
          - 正转7.2S；停止5.2S，反转5.2S，停止3S，正转3S，停止7.2S，反转7.2S，停止3S，正转3S，停止5.2S，反转7.2S，停止7.2S，依此循环工作。
       - 互动/interactive
          - 感应到振动激活一次：正转7.2S；停止5.2S，反转5.2S，停止3S，正转3S，停止5S 旋转期间感应到不激活，结束后感应到则继续激活一次

4、设备状态
- work 优先级中
    - 工作状态，可以进行电机运动
- standby 优先级中
    - 若软件开机时产品工作5分钟后会进入待机模式，指示灯熄灭。当触碰产品后，产品激活继续工作5分钟。
    - 若软件关机时产品只会保持在待机模式不会重新进入工作直到软件开机
- charging 优先级最高
    - 充电状态下产品不工作 
- charge_done 优先级最低
    - 满电推送
